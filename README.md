# OOP-Programs

A collection of Object-Oriented Programming examples and implementations in C++.

## Programs

### AgeSetter.cpp
A simple C++ program demonstrating basic OOP concepts with a class that manages age values.

**Features:**
- `Age` class with member variables and methods
- `display()` method to output age information
- Basic object instantiation and manipulation in `main()`

**How to Compile and Run:**
```bash
g++ AgeSetter.cpp -o AgeSetter
./AgeSetter
```

## About

This repository contains educational OOP programs to demonstrate fundamental object-oriented programming principles including:
- Class definition and structure
- Object creation and initialization
- Member variables and methods
- Encapsulation and data management
